## 📊 Project Overview
This project presents an interactive **Power BI dashboard** built using Uber trip data to analyze ride patterns, customer behavior, and operational performance.

The dashboard helps stakeholders make data-driven decisions using visual analytics.

---

## 🎯 Objectives
- Analyze Uber ride trends
- Identify peak booking hours
- Study trip distance and fare patterns
- Evaluate demand by location
- Create interactive business dashboards

---

## 🛠 Tools & Technologies
- Power BI
- Microsoft Excel / CSV Dataset
- Data Cleaning & Transformation (Power Query)
- Data Visualization

---

## 📈 Dashboard Features
- KPI Cards (Total Trips, Revenue, Distance)
- Trips by Date & Time Analysis
- Pickup Location Analysis
- Fare Distribution
- Interactive Filters & Slicers

---

## 📂 Dataset
Uber trip dataset containing:
- Pickup Date & Time
- Location
- Trip Distance
- Fare Amount
- Passenger Count
